What are breadcrumbs?

Breadcrumbs are persistent tags that are given to tasks when they go through certain execution paths. For example, when a task is created to handle an HTTP request, it is given the HTTP method as a breadcrumb, and the request URL as another one. When you enter a list of breadcrumbs in the textbox, you are specifying which path the task must have had taken first in order to be eligible for the data capture operations.

What are these keywords?

The keywords you are specifying are attached to each piece of data that we will want to capture.

They have been declared in-code this way for parameters: mixin(OnCapture!("HTTP Server Request Parameters", "req.params"))
